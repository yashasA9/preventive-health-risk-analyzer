
# Langflow Pipelines

This folder contains individual Langflow flow exports.

## Files

- model1_risk_analyzer.json  
  Extracts and analyzes user health indicators.

- model2_risk_classifier.json  
  Classifies health risk level and provides preventive guidance.

- model3_rag_qa.json  
  RAG-based follow-up Q&A using verified health documents.

## How to Use
Import each JSON file into Langflow using the "Import Flow" option.
